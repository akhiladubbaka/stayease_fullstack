# 🏨 StayEase – Hotel Booking System

StayEase is a full-stack Hotel Booking System built using the MERN stack. It allows users to browse hotels, search by city, register/login securely, and enables administrators to manage hotel listings through role-based authentication.

---

## 🚀 Live Demo

- **Frontend:** https://stayease-fullstack.vercel.app
- **Backend API:** https://stayease-fullstack-yvng.onrender.com

## ✨ Features

### 👤 Customer
- User Registration
- Secure Login using JWT
- Browse Hotels
- Search Hotels by Name
- Filter Hotels by City
- Filter by Rating
- Filter by Price Range
- View Hotel Details

### 👨‍💼 Admin
- Admin Authentication
- Add Hotels
- Update Hotel Details
- Delete Hotels
- Manage Hotel Listings

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/akhiladubbaka/stayease_fullstack.git
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=8000
MONGO_URI=Your_MongoDB_Connection_String
JWT_SECRET=Your_JWT_Secret
```
